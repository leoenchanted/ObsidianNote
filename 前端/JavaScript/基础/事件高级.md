# 删除事件
```javascript
divs[1].addEventListener('click',fn);
function fn(){
  alert(22);
  divs[1].removeEventListener('click',fn);
}
```

# 事件委托
## 原理
将事件监听器设置在父节点上，然后利用冒泡原理影响设置每个子节点
事件委托的作用： 我们只操作了一次DOM，提高了程序的性能。
