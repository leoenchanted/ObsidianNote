路由懒加载、首屏性能优化、webpack切割代码

```javascript
// 静态导入（不支持懒加载，会打包到主 bundle）
import Home from './Home.vue'; 

// 动态导入（支持懒加载，会生成独立 chunk）
const Home = () => import('./Home.vue');
```

webpack看到import就是相当于懒加载了

**Vite 完全支持通过动态 `import()` 实现路由懒加载**，而且比 Webpack 更简单（无需额外配置）。
